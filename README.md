# clojure-cli-maven-hello-world

## Description
POC for clojure maven build.

## Tech stack
- clojure
- maven

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
